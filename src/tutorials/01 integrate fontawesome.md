## Learning Objective
- [x] Learn to integrate fontawesome in an angular project

## Step 01 - install and update angular.json

```sh
npm install --save @fortawesome/fontawesome-free
```

- update angular.json

```json
"styles": [
    "node_modules/@fortawesome/fontawesome-free/css/all.min.css",
    "src/styles.scss"
]
```
